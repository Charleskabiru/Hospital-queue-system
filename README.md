# 🏥 Hospital Queue Management System

A web-based system designed to manage patient flow from reception to doctors with real-time display and voice announcements.

## 🚀 Features

- 🔐 Role-based login (Admin, Receptionist, Doctor)
- 🧑‍💼 Reception panel for registering patients
- 🧑‍⚕️ Doctor dashboard with department filtering
- 📺 Live display screen with queue updates
- 🔊 Voice announcement system
- 👤 Admin panel for managing users

## 🛠️ Technologies Used

- HTML, CSS, JavaScript
- PHP (Backend API)
- MySQL (Database)
- XAMPP (Local server)

## 📂 System Modules

- Login System
- Reception Dashboard
- Doctor Dashboard
- Display Screen
- Admin Panel

## ⚙️ Setup Instructions

1. Install XAMPP
2. Place project folder in `htdocs`
3. Start Apache & MySQL
4. Import database in phpMyAdmin
5. Open:
   http://localhost/hospital-system/

## 🎯 Purpose

To improve hospital efficiency by reducing waiting time and organizing patient flow digitally.
